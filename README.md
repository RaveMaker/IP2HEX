IP2HEX
======

Convert IP Address to Hex values

### Installation

1. Clone this script from github or copy the files manually to your prefered directory.

2. Run the script and input Subnet or IP Address.

For Example:

- 192.168.0 = c0a800
- 10.0.0 = 0a0000
